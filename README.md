## 🧾 Problem Statement

This project is developed as part of the final assessment.
The system helps manage students, courses, and enrollments efficiently.
It also handles invalid inputs and simulates enrollment processing using multithreading.

---

## 🚀 Features

* Add Student details
* Add Course details
* Enroll Student into courses
* View all Students
* View Enrollments
* Prevent duplicate enrollments
* Exception Handling for invalid inputs
* Multithreading for enrollment processing

---

## 🛠️ Concepts Used

* Java Classes & Objects
* Constructors
* OOP Concepts
* Collections (`HashMap`, `ArrayList`)
* Exception Handling (Custom Exception)
* Multithreading

## ▶️ How to Run
